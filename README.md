# myBEAKER

## 퍼블 작업 시 요청사항
- html tag에 id="" name="" value="" 지정 X 스타일 지정으로 인한 id는 쓰셔도 상관없으나 input 태그에 id="" 이렇게 디폴트 지정은 빼주시기 바랍니다.
- jquery 사용하여 스크립트 작성 X
- 메뉴 left, right show hide 기능 외에 업무화면단 스크립트 동작은 빼주셔도 됩니다.
- TAB 간격은 space 간격 2 로 설정.

## 퍼블 작업 참고 사항
- js 폴더는 메뉴 기능을 위해 추가 하였습니다.
- scss 파일은 css 파일로 컴파일 되었으므로 삭제하셔도 무방합니다.
